Crear clase factura, array de elementos y pseudoclase información.

Propiedades pseudoclase información:baseImponible, iva, total y forma de pago.

Clase Cliente: nombre, dirección, telefono y nif

Clase Elemento: descripción, cantidad y precio

Funciones:Cálculo del total: recorreremos los elementos del array de elementos calculando la base imponible de la factura y el total. Esa información la guardaremos en el total de la pseudoclase información. Base imponible = cantidad * precio. Total = base imponible * (1+ (iva/100)).
